# Mobile Sales Analysis Dashboard
A Power BI dashboard that delivers end-to-end analysis of mobile phone sales performance — tracking revenue, units sold, brand trends, and regional distribution to support data-driven business decisions.

## Project Overview
This project explores mobile sales data through an interactive Power BI report. It helps stakeholders quickly identify top-performing products, monitor sales trends over time, and compare performance across brands, regions, and time periods.

### Repository Contents
FileDescriptionBi_Mobile sales.pbixPower BI Desktop report file containing all data models, transformations, and visualizations

### Key Insights & Features

 Sales Performance Overview — Total revenue, units sold, and average selling price at a glance
 Brand-wise Comparison — Side-by-side analysis of top mobile brands by revenue and volume
 Time-Series Trends — Monthly and quarterly sales trends to identify seasonal patterns
 Regional Breakdown — Geographic distribution of sales across cities/regions
 Top Models — Best-selling mobile models ranked by quantity and revenue
 Interactive Filters — Slicers for brand, model, time period, and region for dynamic exploration

### Getting Started

#### Prerequisites

Power BI Desktop (free download from Microsoft)

Steps to Open

Clone or download this repository:

bash   git clone https://github.com/rt5899-art/Mobile_Sales_Analysis.git

Open Power BI Desktop.
Go to File → Open report and select Bi_Mobile sales.pbix.
Explore the report pages and use the slicers to interact with the data.
